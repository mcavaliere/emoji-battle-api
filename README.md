## Description

[Nest.js](https://github.com/nestjs/nest) API server for the [Emoji Battle](https://github.com/mcavaliere/emoji-battle) project.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
